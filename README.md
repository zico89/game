Space Shooter – 2D Web igra

Space Shooter je jednostavna 2D arkadna igra napravljena korišćenjem Phaser 3 JavaScript biblioteke.
Igrač upravlja svemirskim brodom, uništava neprijatelje, skuplja power-upove i na kraju se bori protiv bossa.

Cilj igre je da igrač osvoji što više poena i pobedi bossa.

Kontrole

Leva / Desna strelica – pomeranje broda

Space – pucanje

Klik mišem – navigacija kroz meni

Funkcionalnosti igre

Glavni meni (Start Game, Options)

Uključivanje i isključivanje zvuka

Različiti tipovi neprijatelja

Neprijateljski projektili

Power-up (Rapid Fire)

Postepeno povećavanje težine

Boss sa health bar-om

Prikaz rezultata (score)

Game Over i restart igre
